## 2. Profil Singkat Website FIKOM UNISAN (Untuk Tim IT / Rektorat)
Penjelasan di bawah ini dapat digunakan sebagai referensi operasional kepada tim pengelola kampus, yayasan, rektorat, maupun pihak teknisi terkait dalam memahami dan mengelola website FIKOM UNISAN Sidrap.

**Apa itu Website FIKOM UNISAN?**
Website FIKOM UNISAN Sidrap adalah portal informasi resmi yang berfungsi sebagai "wajah digital" fakultas. Platform ini menampung dan mendistribusikan segala informasi publik, mulai dari profil jurusan, struktur organisasi fakultas, berita kegiatan kampus, dokumentasi galeri, hingga publikasi tridharma perguruan tinggi (penelitian dan pengabdian masyarakat). Semuanya diatur dan diperbarui melalui satu pintu terpusat menggunakan sistem Admin Dashboard.

**Mengapa Perlu Dilakukan Maintenance Secara Berkala?**
Meskipun website telah berjalan dengan stabil, pemeliharaan (*maintenance*) sistem secara berkala oleh tim IT tetap diwajibkan karena alasan berikut:
1. **Menjaga Kinerja dan Kecepatan Akses:** Memastikan manajemen pengelolaan *server* (Hosting) tetap optimal agar website tidak mengalami kelambatan akses (*loading* lambat) ketika diakses oleh banyak mahasiswa secara bersamaan.
2. **Peningkatan Sistem Keamanan Sistem:** Menutup celah kerentanan kode (*bug*) dan melakukan audit keamanan berkala, agar data publik dan sistem akademik fakultas tidak mudah diretas, dieksploitasi, atau diubah oleh pihak luar.
3. **Pembaruan Visual dan Fungsional:** Mengakomodasi penambahan fitur-fitur teknologi baru atau pembaruan alur informasi agar pendistribusian informasi akademik dapat mengikuti kebutuhan sivitas akademika terkini.
